一、项目介绍
    - 做一个简单的APP自动化测试DEMO，实现一些小功能

二、环境要求
    - 安装python3.5以上
    - 安装Appium-desktop：1.15.1以上，安装selenium库、appium库等等

三、框架介绍
    - Common：存放封装的公共类
    - Outputs：存放输出的日志，报告，截图等
    - PageLocators：存放各页面的元素
    - PageObjects：存放各页面的操作对象
    - TestCases：存放测试用例
    - TestDatas：存放测试数据
    - run.py：运行文件

四、项目维护
    - 1.0版本：构建测试框架及整理思路
